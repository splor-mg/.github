# Código de Conduta

Este documento define os padrões de conduta esperados para todos os contribuidores
do trabalho desenvolvido neste repositório.


## Abrangência

Aplica-se a todos que de alguma forma contribuírem para este repositório — por
issues, pull requests, revisões de código ou qualquer outra forma de participação —
independentemente de vínculo institucional.

## Princípios

A convivência neste repositório deve ser guiada por respeito, honestidade,
transparência, qualidade técnica e colaboração.

## Condutas esperadas

- Comunicar-se com respeito e objetividade em todas as interações
- Fundamentar decisões técnicas com clareza, permitindo revisão e discussão aberta
- Aceitar revisões de código como parte natural do processo colaborativo, não como
  julgamento pessoal
- Reportar problemas de segurança ou inconsistências de forma responsável, pelos
  canais adequados
- Zelar pela qualidade e rastreabilidade das contribuições, em conformidade com o
  `CONTRIBUTING.md`

## Condutas vedadas

- Discriminação ou assédio de qualquer natureza
- Linguagem ofensiva, intimidadora ou desrespeitosa
- Atribuição de autorias indevidas ou omissão intencional de contribuições alheias;
  consulte as especificações contidas no arquivo `LICENSE.md`
- Manipulação de histórico ou registros para ocultar erros ou responsabilidades

## Diretrizes de Consequências

Os mantenedores seguirão estas diretrizes ao determinar as consequências para ações
que violem este Código[^1]. As Diretrizes de Consequências foram inspiradas pelo
[código de conduta da Mozilla](https://github.com/mozilla/diversity)[^2].

**1. Correção**
Uso de linguagem inapropriada ou comportamento considerado não profissional.
*Consequência:* aviso escrito e privado, com explicação sobre a violação. Uma
retratação pública poderá ser solicitada.

**2. Aviso**
Violação por incidente único ou série de ações.
*Consequência:* advertência formal com restrição de interação com as pessoas
envolvidas por período determinado.

**3. Banimento Temporário**
Violação grave ou comportamento inapropriado recorrente.
*Consequência:* suspensão temporária de qualquer interação pública no repositório.

**4. Banimento Permanente**
Padrão sistemático de violações, assédio ou agressão.
*Consequência:* exclusão permanente de qualquer participação no repositório.

[^1]: Este Código de Conduta é adaptado do [Contributor Covenant](https://www.contributor-covenant.org),
versão 2.0, disponível em https://www.contributor-covenant.org/version/2/0/code_of_conduct.html.

[^2]: Mozilla's code of conduct enforcement ladder: https://github.com/mozilla/diversity.
